# PDF 转 PNG 图片工具

> 📅 项目创建日期：2026-02-27

一个简洁高效的 PDF 转 PNG 在线转换工具，支持批量转换和打包下载。

## 功能特性

- 📁 支持拖拽上传 PDF 文件
- 🔢 自动按页码生成 PNG 图片（page_1.png, page_2.png...）
- 📐 支持 1x-4x 多种图片清晰度选择
- 🖼️ 实时预览转换结果
- 📥 一键打包下载为 ZIP 文件

## 使用方法

1. 直接用浏览器打开 `pdf-to-png.html` 文件
2. 拖拽或点击选择 PDF 文件
3. 选择图片清晰度（默认 2x 高清）
4. 点击"开始转换"
5. 转换完成后点击"打包下载所有图片"

## 技术栈

- PDF.js - PDF 解析和渲染
- JSZip - ZIP 文件打包
- 纯前端实现，无需服务器

## 浏览器支持

- Chrome / Edge / Firefox / Safari (现代浏览器)
