# PDF 工具箱

> 📅 项目创建日期：2026-02-27
> 🕐 最后更新：2026-02-27

一个简洁高效的 PDF 处理工具，支持两种模式：页面转图片和提取嵌入图片。

## 功能特性

### 🖼️ 模式一：页面转图片
- 📁 支持拖拽上传 PDF 文件
- 🔢 每页单独生成一张 PNG 图片
- 📐 支持 1x-4x 多种图片清晰度选择
- 📥 一键打包下载为 ZIP 文件

### 📸 模式二：提取图片
- 🔍 智能识别 PDF 中嵌入的图片（插图、照片等）
- ⏭️ 自动跳过只有文字的页面
- 📋 按页码和提取顺序编号（如：img_1_1.png, img_1_2.png, img_2_1.png...）
- 🖼️ 支持原始质量/高/中/低 多种输出质量
- 📥 一键打包下载

## 使用方法

1. 用浏览器打开 `pdf-to-png.html`
2. 选择模式：
   - **页面转图片**：将 PDF 每页转为图片
   - **提取图片**：从 PDF 中提取嵌入的图片
3. 拖拽或点击选择 PDF 文件
4. 点击"开始转换"或"开始提取图片"
5. 转换完成后点击"打包下载所有图片"

## 技术栈

- **PDF.js** - PDF 解析和渲染
- **JSZip** - ZIP 文件打包
- 纯前端实现，无需服务器

## 浏览器支持

- Chrome / Edge / Firefox / Safari (现代浏览器)
